# ATlan Tech

## Descripción
Tienda en línea donde el cliente puede comprar productos sin necesidad 
de iniciar sesión. Cuenta con una sección de administración protegida 
para gestionar el catálogo de productos.

## Funciones para el cliente
- Ver los productos disponibles
- Agregar productos al carrito
- Editar su pedido (agregar o quitar productos)
- Llenar sus datos y realizar el pedido

## Funciones del administrador
- Iniciar sesión en el panel admin
- Ver todos los productos
- Crear nuevos productos
- Editar productos existentes
- Eliminar productos

## Tecnologías usadas
- Python / Flask
- SQLAlchemy (base de datos)
- Flask-Login (autenticación)
- HTML5
- CSS3

## Cómo ejecutar el proyecto
1. Activar el entorno virtual: `venv\Scripts\activate.bat`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Correr la aplicación: `python run.py`
4. Abrir en el navegador: `http://127.0.0.1:5000/`
